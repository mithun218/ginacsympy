Mathematical function

All predefined mathematical functions in GinacSympy are all objects of class function. They accept one or more expressions as arguments and return one expression. If the arguments are not numerical, the evaluation of the function may be halted, as it does in the next example. 

 >>>(sin(x)).is_a(function)
 True
 >>>gamma(x)
 gamma(x)
 >>>gamma(3)
 6

Besides evaluation most of these functions allow differentiation, series expansion and so on. There are quite a number of useful functions hard-wired into GinacSympy. The predefined mathematical functions are

  1. Numerical Functions

    2. conjugate_function,  real_part_function  imag_part_function,  abs,  csgn, 

  2. Combinatorial Functions

    3. factorial,  binomial 

  3. Elementary Transcendental Functions

    4. exp,  log,  logb,  eta, 

  4. Trigonmetric Functions

    5. sin,  cos,  tan,  csc,  sec,  cot, 

  5. Hyperbolic Functions

    6. sinh,  cosh,  tanh,  csch,  sech,  coth, 

  6. Inverse Functions

    7. asin,  acos,  atan,  acsc,  asec,  acot,  asinh,  acosh,  atanh,  acsch,  asech,  acoth, 

  7. Orthogonal Polynomials

    8. legendreP,  hermiteH,  gegenbauerC,  chebyshevT,  chebyshevU, 

  8. Special Functions

    9. legendreQ,  besselJ,  besselY,  bessel,  besselK, 

  9. Elliptic Integrals and Elliptic Functions

    10. EllipticK,  EllipticE, 

  10. Exponential Integral and Related Functions

    11. sinIntegral,  cosIntegral,  sinhIntegral,  coshIntegral,  expIntegralE,  expIntegralEi,
