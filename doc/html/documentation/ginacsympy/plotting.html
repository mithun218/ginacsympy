Plotting

to_numpy_array,

GinacSympy expressions can be plotted using matplotlib library. In order to plot, at first we have to convert the GinacSympy expressions into numpy array using the following function

 to_numpy_array(Ex f, list var_list, list array_list)

Here f is the GinacSympy expression, var_list is the variable list of coordinate axes, and array_list is the corresponding numpy array of the variable list. The use of this function is cleared by the following examples:

2d example: 
 >>> import numpy as np
 >>> import matplotlib.pyplot as plt
 >>> from ginacsympy_abc import x
 >>> from ginacsympy import sin,Pi,to_numpy_array
 >>> f=5*sin(x*Pi.evalf())
 >>> x_a=np.linspace(0,5,100)
 >>> y=to_numpy_array(f,[x],[x_a])
 >>> plt.plot(x_a,y)
3d example: 
 >>> import numpy as np
 >>> import matplotlib.pyplot as plt
 >>> from ginacsympy_abc import x,y
 >>> from ginacsympy import sin,Pi,to_numpy_array
 >>> ax = plt.axes(projection='3d')
 >>> f=x**2+y**2
 >>> x_a=np.linspace(0,5,100)
 >>> y_a=np.linspace(0,5,100)
 >>> f_a=to_numpy_array(f,[x,y],[x_a,y_a])
 >>> ax.plot(x_a,y_a,f_a)
