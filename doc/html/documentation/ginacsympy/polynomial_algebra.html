is_polynomial(Ex var):

Testing whether an expression is a polynomial in one or more variables can be done with this method. 
 >>> (x*y+x*x+sin(y)**2).is_polynomial(x)
 True
 >>> (x*y+x*x+sin(y)**2).is_polynomial(lst([x,y]))
 False

degree(Ex var):

The degree of a polynomial with respect to variable var can be obtained using this method.

 >>> (x*y+x*x+y**2).degree(y)
 2

ldegree(Ex var):

The low degree of a polynomial with respect to variable var can be obtained using the this method.

 >>> (x*y+x*x*y**3+y**2).ldegree(y)
 1

coeff(Ex var, int n):

To extract a coefficient of the variable var with a certain power n from an expanded polynomial you can use this method. 

 >>> (x*y+x*x*y**3+y**2).coeff(y)
 x

lcoeff(Ex var):

You can also obtain the leading coefficient of the variable varfrom an expanded polynomial you can use this method.

 >>> (x*y+x*x*y**3+y**2).lcoeff(y)
 x**2

tcoeff(Ex var):

You can also obtain the trailing coefficient of the variable varfrom an expanded polynomial you can use this method.

 >>> (x*y+x*x*y**3+y**2).tcoeff(y)
 x

Ex quo(Ex a, Ex b, Ex x):

It computes quotient of the polynomial a in the variable x. The results satisfy a = b*quo(a, b, x) + rem(a, b, x).

 >>> quo(a*x**2+2*a*x*y+a*y**2+b,x+y,x)
 a*x+a*y

Ex rem(Ex a, Ex b, Ex x):

It computes remainder of the polynomial a in the variable x. The results satisfy a = b*quo(a, b, x) + rem(a, b, x).

 >>> rem(a*x**2+2*a*x*y+a*y**2+b,x+y,x)
 b

Ex prem(Ex a, Ex b, Ex x):

This function computes the pseudo-remainder of a and b satisfies c*a = b*q + prem(a, b, x), where c = b.lcoeff(x)(a.degree(x) - b.degree(x) + 1).

 >>> prem(a*x**2+2*a*x*y+a*y**2+b,x+y+a,x)
 a**3+b

divide(Ex a, Ex b):

Exact division of multivariate polynomials is performed by this function. If b divides a over the rationals, this function returns True and returns the quotient in a tuple. Otherwise it returns False in which case the value of quotient is undefined.

 >>> divide(x**2+2*x*y+y**2,x+y)
 (True,x+y)
 >>>  divide(a*x**2+2*x*y+y**2,x+y)
 (False,)

unit(Ex x):

It returns the unit partof a multivariate polynomial with respect to the variable x (the unit part is the sign of the leading coefficient).

 >>> (a*y*x**2+2*x*y+y**2).unit(x)
 1

content(Ex x):

It returns the content part of a multivariate polynomial with respect to the variable x (the content part is the GCD of the coefficients).

 >>> (a*y*x**2+2*x*y+y**2).content(x)
 y

primpart(Ex x, Ex c=None):

It returns the primitive polynomial of a multivariate polynomial with respect to the variable x (the primitive polynomial being the input polynomial divided by the unit and content parts). The product of unit, content, and primitive part is the original polynomial. The second argument c expects the previously calculated content part of the polynomial, which enables it to work faster in the case where the content part has already been computed.

 >>> (a*y*x**2+2*x*y+y**2).primpart(x)
 y+2*x+x**2*a

unitcontprim(Ex x)

It computes the unit u, content c, and primitive part p in one go, returning them in a tuple (u,c,p).

 >>> (a*y*x**2+2*x*y+y**2).unitcontprim(x)
 (1,y,y+2*x+x**2*a)

gcd(list l):

This function accept list of expressions as arguments and return a new expression, their greatest common divisor.

 >>> gcd([x**2-1,x-1,x-1,x**3-x**2])
 -1+x

lcm(list l):

This function accept list of expressions as arguments and return a new expression, their least common multiple.

 >>> lcm([x**2-1,x-1,x-1,x**3-x**2])
 (-1+x)*x**2*(1+x)

resultant(Ex a, Ex b, Ex s):

The resultant of two polynomials a and b. It is always computed with respect to a specific symbol s within the expressions.

 >>> resultant(x**4+15*x*y+56,x**4*y+12*y+12,x)
 20736-304128*y+3748096*y**4+607500*y**7+1672704*y**2-4088832*y**3+607500*y**8

factor(Ex a, factor_options opt=factor_options_polynomial):

Polynomial a can also be fully factored with a call to this function. The factorization works for univariate and multivariate polynomials with rational coefficients.

factor_options: It has two options.

factor_options_polynomial-> This is due to the default option (equals zero), which tells factor to try a factorization only if the expression is a valid polynomial.

factor_options_all-> It tells factor to ignore non-polynomial parts of an expression and also to look inside function arguments.

 >>> factor(expand((a+b+c+d)**20))  
 (a+b+c+d)**20
