  1. Limit:

    2. Limit points are sustituted for non-critical points. L’Hôpital’s rule is used in critical cases, with preprocessing of ∞/∞, 0/0, ∞-∞, 1^∞, 0^0, ∞^0 forms and reformatting of product forms in order to apply L’Hôpital’s rule. A limited amount of bounded arithmetic is also employed where applicable.

    limit(e, Ex z, z0, str dir="+-"):

    e : expression, the limit of which is to be taken

    z : symbol representing the variable in the limit. Other symbols are treated as constants. Multivariate limits are not supported.

    z0 : the value toward which z tends. Can be any expression, including ∞ and -∞.

    dir : string, optional (default: "+-"). If the limit depends upon the direction of approach to the limpoint, the onesided limit can be calculated using this string. The limit is bi-directional if dir="+-", from the right (z→z0+) if dir="+", and from the left (z→z0-) if dir="-". For infinite z0 (∞ or -∞), the dir argument is determined from the direction of the infinity (i.e., dir="-" for ∞). 

     >>> limit(2/(x-3),x,3,"-")
 -Infinity
 >>> limit(2/(x-3),x,3,"+")
 Infinity
 >>> limit(2/(x-3),x,3)
 Limit(2*(-3+x)**(-1),x,3)
 >>> limit(exp(3*x)/(exp(2*x)*log(x)+3),x,-Infinity)
 0
 >>> limit((exp(x)+exp(4*x)*log(x))/(exp(x)+exp(6*x)),x,0)
 Infinity

    Limit(e, Ex z, z0, str dir="+-"):

    This class is the Inert form of limit.

     >>> Limit(2/(x-3),x,3,"+")
 Limit(2/(x-3),x,3,"+")

    expand():

    Apply addition rule of Limit.

    >>> Limit(x**2+1/(1+x),x,a,"+").expand()
Limit(-2*(z-x)**(-1),x,z,+)

    subs(dict m, subs_options opt=subs_nooptions):

    Substitute m in the first and third operands of Limit.

     >>> Limit(2/(x-a),x,a,"+").subs({a:z})
 Limit(-2*(z-x)**(-1),x,z,+)

    evaluate():

    Evaluate inert form of limit.

     >>> Limit(2/(x-3),x,3,"+").evaluate()
 Infinity

  2. Differentiation

    3. diff(Ex x, int n = 1):

    It computes n-th order differentiation or partial differentiation of an expression with respect to x. 

     >>> (x**n).diff(x,2)
 n*(n-1)*x**(n-2)

    Diff(Ex x, int n = 1):

    This class is the inert form of diff.

     >>> Diff((sin(x)*a*x+x),x,2)
 Diff((sin(x)*a*x+x),x,2)

    expand():

    Apply addition rule of Diff.

    >>> Diff(x**2+1/(1+x),x,2).expand()
 Diff(x**2,x,2)+Diff(1/(1+x),x,2)

    subs(dict m, subs_options opt=subs_nooptions):

    Substitute m in the first operand of Diff.

     >>> Diff(x**2-a**2,x).subs({a**2:z})
 Diff(-z+x**2,x,1)

    change_variable(Ex oldNewOrNewOld, Ex newvarName):

    We can change differentiation variable of Diff class.

    oldNewOrNewOld is a relational object that contains the transformation equation from old variable to new variable, or new variable to old variable.

     >>> f = (Diff(x**2+x**4,x,2)).change_variable(relational(z,x**2),z)
 >>> f
 2*Diff(2*x*Diff(x**4+x**2,z,1),z,1)*x
 >>> f=f.subs({x:sqrt(z)})
 >>> f
 2*Diff(2*z**(1/2)*Diff(z*(1+z),z,1),z,1)*z**(1/2)
 >>> f=f.evaluate().simplify()
 >>> f
 2+12*z

    evaluate():

    Evaluates inert Diff.

     >>> Diff((sin(x)*a*x+x),x,2).evaluate()
 -4*a*x*cos(x)*sin(x)

  3. Integration

    4. Integration of an algebraic expression is performed by partial integration and pattern matching.

    integrate(Ex expr, Ex var, partial_num=-1):

    Gives indefinite integration of algebraic expression expr with respect to variable var. partial_num is the number of step of partial integration. By default it is assigned to -1 and GinacSympy can automatically detect the termination of partial integration. Unfortunately, GinacSympy is not so matured to integrate an algebraic expression. Sometime it cann't detect the infinite loop of partial integration and so it is desired to quote the number of step of partial integration.

     >>> integrate((x+a)*exp(a*x+b),x)
 exp(x*a+b)+exp(x*a+b)*x*a**(-1)-exp(x*a+b)*a**(-2)

    integrate(Ex expr, Ex var,l, u, partial_num=-1):

    Gives definite integration of algebraic expression expr with respect to variable var between the lower limit l and upper limit u.

     >>> integrate((x+a)*exp(a*x+b),x,0,1) 
 exp(a+b)*a**(-1)-exp(b)+exp(a+b)-exp(a+b)*a**(-2)+exp(b)*a**(-2)

    Integrate(Ex expr, Ex var, partial_num=-1):

    It is inert form of indefinite integrate.

     >>> Integrate((x+a)*exp(a*x+b),x)
 Integrate((x+a)*exp(a*x+b),x)

    Integrate(Ex expr, Ex var,l, u, partial_num=-1):

    It is inert form of definite integrate.

     >>> Integrate((x+a)*exp(a*x+b),x,0,1)  
 Integrate((x+a)*exp(a*x+b),x,0,1)

    expand():

    Apply addition rule of Integrate.

     >>> Integrate(x**2+1/(1+x),x,2).expand()
 Integrate((1+x)**(-1),x)+Integrate(x**2,x)

    set_partial_num(int i):

    It sets number of steps of partial integration of Integrate object, and when its value is evaluated by evaluate method, this number is used by integrate method.

     >>> f=Integrate(sin(b*x)*log(x),x)
 >>> f.set_partial_num(2)
 >>> f.evaluate()
 Integrate(x**(-2)*sin(x*b),x)*b**(-2)-cos(x*b)*log(x)*b**(-1)+x**(-1)*b**(-2)*sin(x*b)
 >>> f.set_partial_num(3)
 >>> f.evaluate()
 -cos(x*b)*x**(-2)*b**(-3)-cos(x*b)*log(x)*b**(-1)-2*Integrate(cos(x*b)*x**(-3),x)*b**(-3)+x**(-1)*b**(-2)*sin(x*b)

    subs(dict m, subs_options opt=subs_nooptions):

    Substitute m in the first operand of Integrate.

     >>> Integrate(x**2-a**2,x).subs({a**2:z})
 Integrate(x**2-z,x)

    apply_partial_integration(int partial_num):

    This method of Ex object apply partial_num number of partial integration on Integrate object.

     >>> f=Integrate(sin(b*x)*exp(x),x)
 >>> f=f.apply_partial_integration(1)
 >>> f
 -cos(x*b)*exp(x)*b**(-1)+b**(-1)*Integrate(cos(x*b)*exp(x),x)
 >>> f=f.apply_partial_integration(2)
 >>> f
 -cos(x*b)*exp(x)*b**(-1)+(exp(x)*b**(-1)*sin(x*b)-Integrate(exp(x)*sin(x*b),x)*b**(-1))*b**(-1)
 >>> f.diff(x)
 sin(b*x)*exp(x)

    change_variable(Ex oldNewOrNewOld, Ex newvarName):

    We can change integration variable of Integrate class.

    oldNewOrNewOld is a relational object that contains the transformation from old variable to new variable, or new variable to old variable.

     >>> f = Integrate(1/(1+sin(x)),x).change_variable(relational(z,sin(x)),z)
 >>> f
 Integrate(sec(x)*(1+sin(x))**(-1),z)  
 >>> (f).subs({sec(x):1/sqrt(1+z**2)})
 Integrate((1+z)**(-1)*(1+z**2)**(-1/2),z)
 >>> f = (Integrate(cos(x)/(1+sin(x)),x,0,Pi/Ex(2))).change_variable(relational(z,1+sin(x)),z)
 >>> f
 Integrate((1+sin(x))**(-1),z,1,2)
 >>> f.subs({1+sin(x):z})
 Integrate(z**(-1),z,1,2)

    evaluate():

    It evaluates the inert form Integrate.

    >>> Integrate((x+a)*exp(a*x+b),x,0,1).evaluate()
exp(a+b)*a**(-1)-exp(b)+exp(a+b)-exp(a+b)*a**(-2)+exp(b)*a**(-2)
