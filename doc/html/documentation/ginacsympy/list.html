Lists of expressions

The GinacSympy class lst serves for holding a list of arbitrary expressions. 

lst,
Member functions of lst class: 

add(),
mul(),
prepend() and append(),
remove_first(), remove_last(), and remove_all(),
sort(),
unique(),

lst:

lst class can be constructed from a python list of expressions: 

 >>>li=lst([x*x,y+z,x])
 >>>li
 {x**2,y+z,x}

Use the nops() method to determine the size (number of expressions) of a list and the op() method or the [] operator to access individual elements: 

 >>>li.nops()
 3
 >>>li.op(0)
 x*x
lst allows in-place modifications. You can modify single element using [] operator: 
 >>>li[1]=x*z
 >>>li
 {x**2,x*z,x}

You can iterate over the lst class:

 >>>li=lst([x*x,y+z,x])
 >>>for i in li:
        print(i)
 x**2
 y+z
 x
 >>>for i in reversed(li):
        print(i)
 x
 y+z
 x**2
The member functions of Ex object subs,simplify,diff,integrate can be applied to each element of lst object: 
 >>>li.diff(x)
 {2*x,0,1}
 >>>li.integrate(x)
 {x**3/3,(y+z)*x,x**2/2}

add():

It sums all elements of lst class.

 >>>li.add()
 x**2+y+z+x

mul():

It multplies all elements of lst class.

 >>>li.mul()
 x**3*(y+z)

prepend() and append():

You can prepend or append an expression to a lst with the prepend() and append() methods:

 >>>li.prepend(y)
 >>>li
 {y,x*x,y+z,x}
 >>>li.append(z)
 >>>li
 {y,x*x,y+z,x,z}

remove_first(), remove_last(), and remove_all():

You can remove the first or last element of a list with remove_first() and remove_last():

 >>>li=lst([x,y,z])
 >>>li.remove_first()
 >>>li
 {y,z}
 >>>li.remove_last()
 >>>li
 {y}

You can remove all the elements of a list with remove_all():

 >>>li.remove_all()
 >>>li
 {}

sort():

You can bring the elements of a lst into a canonical order with sort(): 

 >>>l1 = lst([x, 2, y, x+y])
 >>>l2 = lst([2, x+y, x, y])
 >>>l1.sort();
 >>>l2.sort();# l1 and l2 are now equal
 >>>l1
 {y,y+x,2,x}
 >>>l2
 {y,y+x,2,x}

unique():

Finally, you can remove all but the first element of consecutive groups of elements remain in the list with unique(): 

 >>>l3 = {x, 2, 2, 2, y, x+y, y+x}
 >>>l3.unique();        
 >>>l3 
 {x, 2, y, x+y}
