Mathematical Constants

Constants behave pretty much like symbols except that they return some specific number when the method .evalf() is called. These constants are all objects of class constant.

The predefined known constants are:

NameCommon NameNumerical Value (to 35 digits)
PiArchimedes’ constant3.14159265358979323846264338327950288
CatalanCatalan’s constant0.91596559417721901505460351493238411
EulerEuler’s (or Euler-Mascheroni) constant0.57721566490153286060651209008240243
 >>>Pi.evalf()
 3.1415926535897932385
 >>>set_digits(30) # with 30 decimal digits precision
 >>>Pi.evalf()
 3.1415926535897932384626433832795028842
