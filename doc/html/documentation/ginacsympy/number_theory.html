Number Theory

mod,   smod,   irem,   iquo,   quotient_remainder,   divisible,   is_prime,   base_form,   next_prime,   random,   random_prime,

mod(a,b):

Modulus (in positive representation). In general, mod(a,b) has the sign of b or is zero, and rem(a,b) has the sign of a or is zero. This is different from Maple's modep, where the sign of b is ignored. It is in agreement with Mathematica's Mod. It returns a mod b in the range [0,abs(b)-1] with sign of b if both are integer, 0 otherwise. 
 >>> mod(13,-3)
 -2

smod(a,b):

Modulus (in symmetric representation). It returns a mod b in the range [-iquo(abs(b),2), iquo(abs(b),2)]. 
>>> smod(13,-3)
-2

irem(a,b):

Numeric integer remainder. Equivalent to Maple's irem(a,b) as far as sign conventions are concerned. In general, mod(a,b) has the sign of b or is zero, and irem(a,b) has the sign of a or is zero. 
 >>> irem(5,-2)
 1

iquo(a,b):

Numeric integer quotient. Equivalent to Maple's iquo as far as sign conventions are concerned. return truncated quotient of a/b if both are integer, 0 otherwise. 
 >>> iquo(5,-2)
 -2

quotient_remainder(a,b):

Numeric integer quotient and remainder. Equivalent to Maple's iquo(a,b,'r') it obeyes the relation r == a - iquo(a,b)[0]*b. It returns truncated quotient of a/b and remainder if both are integer, 0 otherwise. 
 >>> quotient_remainder(5,-2)
 {-2,1}

divisible(a,b):

Yields True if b is divisible by a, and yields False if it is not. 
 >>> divisible(5,2)
 False

is_prime(a):

Test whether a is prime. 
 >>> is_prime(5)
 True

base_form(a,b):

It converts the decimal number a in base b. 
 >>> base_form(23,2)
 10111

next_prime(a):

Gives the smallest prime above a. 
 >>> next_prime(23)
 29

random(a):

a must be an integer > 0. This function returns a pseudorandom integer x in the range 0 <= x < a. 
 >>> random(23)
 21

random_prime(a):

gives a pseudorandom prime number in the range 2 to a. 
>>> random_prime(23)
19
