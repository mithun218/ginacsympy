Contributions and bug reports

Contributions to this project are very welcome. If you wish to contribute a new feature, you can do this by forking the ginacsympy repo on github page and creating a branch. Apply your code changes to the branch on your fork. When you're done, submit a pull request to merge your fork into master branch with a tag "enhancement", and the proposed changes can be discussed there. If you encounter a bug, please open a new issue on the GitHub repository to report the bug, and tag it "bug". Please provide sufficient information to reproduce the bug and include as much information as possible that can be helpful for fixing it.
